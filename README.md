# aos-sqlite-vec

A SQLite vector database module for ao computer.

## Prerequisites

- Node.js
- Arweave wallet
- EMSDK
- Make
- Docker
- pnpm

## Build

```
make vendor
make build

pnpm install
```

## Deploy a module

```
node script/deploy_module.js
```
